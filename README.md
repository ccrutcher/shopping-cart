A React project designed to create a mock online shopping cart.

Through the use of React's router, users are first brought to a landing page that shares this fake company's mission statement.

![Pet2](https://user-images.githubusercontent.com/35352111/121033265-a8134880-c771-11eb-9a2d-53005e4a8649.png)

Upon going to the shop, users see the list of 'product' (pets to be pet).

![Pet3](https://user-images.githubusercontent.com/35352111/121033339-b95c5500-c771-11eb-937c-e7a52493bb72.png)

Users can then select how many pets they would like to buy, which is then rendered on the right side of the page.

![Pet1](https://user-images.githubusercontent.com/35352111/121033458-d55ff680-c771-11eb-93df-5c565cbebfac.png)
